# B-Tree Implementation in C++

## Objective

To implement a B-Tree in C++ with insertion, search, and traversal operations.

## Features

- Interactive menu-driven program
- Insert keys into B-Tree
- Search for values
- Display traversal
- Automatic node splitting

## Compilation

```bash
g++ btree.cpp -o btree
```

## Execution

```bash
./btree
```

## Operations Supported

1. Insert
2. Search
3. Traverse
4. Exit

## Concepts Used

- Balanced multi-way tree
- Key insertion
- Node splitting
- Recursive search
- Traversal

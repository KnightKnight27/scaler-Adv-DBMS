## Module Descriptions

| Module | Files | Role |
|---|---|---|
| Storage | `disk_manager`, `buffer_pool`, `heap_file`, `page` | Persist rows to disk |
| Index | `bplus_tree` | O(log n) primary-key lookup |
| Query | `parser`, `optimizer`, `executor` | Parse SQL вҶ’ plan вҶ’ execute |
| Transaction | `lock_manager`, `tx_manager` | 2PL, begin/commit/abort |
| Recovery | `wal`, `recovery` | WAL append; crash-recovery redo |
| Replication | `primary`, `replica` | Log-shipping, Track D |

## Project Overview

Capstone project for Advanced DBMS: build a small relational database (MiniDB) with storage, indexing, query execution, transactions, and recovery.

**Status:** Project started. Implementation in progress.

---

## Planned Modules

- Storage (page manager, buffer pool, heap files)
- B+ tree indexing
- SQL parser and executor (SELECT, INSERT, DELETE, WHERE, JOIN)
- Cost-based optimizer
- Transactions (2PL, serializable)
- WAL and crash recovery
- One extension track

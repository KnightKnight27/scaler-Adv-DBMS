## About our Project
Our project implements **MiniDB**, a high-performance relational database engine playground implementing advanced DBMS internals: slotted-page storage, B+ Tree indexing, Volcano-style physical execution, cost-based optimization, PostgreSQL-style Multi-Version Concurrency Control (MVCC), and ARIES crash recovery. It features a state-of-the-art interactive web dashboard for real-time visualization.

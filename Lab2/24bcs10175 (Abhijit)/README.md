This assignment contains SQLite3 and PostgreSQL shell scripts for basic database operations and comparison.

Files included:
- sqlite_lab.sh
- psql_lab.sh
- observations.md

The scripts perform:
- table creation
- data insertion
- query execution
- page size and page count analysis
- mmap_size analysis
- query timing comparison

The scripts were tested in Linux/WSL environment using bash.

Note:
`psql_lab.sh` uses Linux-specific PostgreSQL authentication (`sudo -u postgres`) and is intended to run in Linux/WSL environment.

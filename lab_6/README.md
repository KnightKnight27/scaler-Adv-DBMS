# B-Tree Implementation in C++

## Objective

To implement a B-Tree data structure in C++ with insertion, search, and traversal operations.

## Features

- Insert elements
- Search for a value
- Traverse tree contents
- User-defined minimum degree

## Compilation

```bash
g++ btree.cpp -o btree
```

## Execution

```bash
./btree
```

## Sample Operations

- Insert values
- Search existing/non-existing values
- Print traversal

## Concepts Used

- B-Tree nodes
- Node splitting
- Recursive traversal
- Search path navigation
